# web
Web project MVP


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```

### generate icons
```
yarn icons
```

### Run vue cli ui
```
yarn ui
```
